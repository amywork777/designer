## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/amzyst/createy.git
   cd createy
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3002](http://localhost:3002) in your browser.
